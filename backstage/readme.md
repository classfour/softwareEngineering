# 后台管理系统
## 简介
- 本系统为后台管理员登陆，主要包括数据维护，开放选课，发布通知，信息查询等功能。

## 开发环境选择
- 本班原采用Java作为开发语言，后端框架为springboot，上手难度较高。因为后台管理几乎与前台操作独立，故在与团长商量，组员开会后，采用较简单python的flask作为开发框架。
